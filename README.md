---

```
- Clone esse repositório:
$ git clone https://github.com/valdir-alves3000/ignite-lab
- Entre no diretório:
$ cd ignite-lab
- Instale as dependências:
$ npm install
# or
yarn install
- Inicie o app:
$ npm run dev
# or
yarn dev
```
